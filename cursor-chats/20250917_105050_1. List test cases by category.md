### 2. **Flexible Test Execution**
Users can run tests in multiple ways:
- **Interactive mode** - Browse and select tests through a menu
- **Direct execution** - Run specific tests by ID (`--run mem-1`)
- **Category execution** - Run all tests in a category (`--category memory`)
- **List mode** - View all available tests (`--list`)

### 3. **Smart API Key Handling**
- Tests requiring API keys are clearly marked
- Tests without API requirements can run immediately
- The runner automatically detects if `MOONSHOT_API_KEY` is configured
